## 快速开始路径

```
你是谁？
  │
  ├── 个人用户 ──→ 安装 OpenClaw ──→ 绑定 Telegram ──→ 配置早间简报 Cron ──→ 逐步添加技能
  │
  ├── 开发者 ──→ 安装 OpenClaw ──→ 配置 GitHub MCP ──→ 设置 PR 审查 Webhook ──→ 添加 CI 监控
  │
  └── 企业团队 ──→ VPS/Docker 部署 ──→ 安全加固 ──→ Staging 验证 ──→ 选择业务场景逐步落地
```

| 你的需求 | 推荐硬件 | 预算 | 说明 |
